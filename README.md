Meme Sentiment Review Engine

This project is about analyzing the sentiment of memes using machine learning. The system studies meme content and predicts whether the sentiment expressed in the meme is positive, negative, or neutral.
Project Overview
Memes are widely used on social media to express opinions and emotions. In this project, a machine learning model is used to analyze meme content and determine the sentiment behind it.

Technologies Used

- Python
- Jupyter Notebook
- Machine Learning
- Natural Language Processing (NLP)

Project Files

- Main.ipynb – Contains the implementation of the meme sentiment analysis model.
- REPORT.pdf – Contains the detailed explanation and documentation of the project.
- H4_minor_project.pptx – Contains the presentation slides of the project.

How to Run the Project

1. Open the project in Jupyter Notebook.
2. Run the "Main.ipynb" file step by step.
3. Follow the instructions in the notebook to analyze meme sentiment.
